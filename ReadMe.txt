Le zip "CatPlayer.zip" est à importer avec Android Studio pour reconstruire les fichiers de compilation.
Il a été conçu sur la version 25 d'Android.

Le fichier CatPlayer.apk est le fichier d'installation de l'application sur téléphone.
Il ne faut pas oublier de donner à l'application l'autorisation d'accès au stockage du téléphone.

Le rapport est en deux fichiers :
- le fichier principal est "Rapport CatPlayer.pdf"
- l'autre fichier est un fichier pdf A3 contenant l'architecture du projet "Architecture_CatPlayer.pdf"
